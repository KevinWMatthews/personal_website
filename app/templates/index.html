About

Kevin here - embedded developer by day, aspiring polyglot by night.

My daily work is on embedded Linux systems in C, with a web interface in Python and JavaScript I've also used and studied C++, Ruby, and Rust.

I'm hooked on fast feedback loops, automated tests, and solutions that are designed to evolve. I strive to craft thoughtful, clean, testable code because I like to be proud of what I deploy.

Projects arrow_downward

Projects

I have developed professionally for bare-metal, FreeRTOS, and embedded Linux devices and have built a custom Linux distro using Buldroot. These systems include VoIP endpoints, RFID readers (NFC), and water chemistry controllers.

All of this work is proprietary, so I'm creating a blog to share projects that I've made on the side.
